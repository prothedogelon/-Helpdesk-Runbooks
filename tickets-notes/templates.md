## No Internet (Example)
**Issue:** User reports no internet while connected to Wi-Fi.  
**Impact:** User unable to access web apps/email.  
**Environment:** Mac / Wi-Fi / VPN: No  
**Troubleshooting performed:** Checked scope, verified Wi-Fi SSID, confirmed IP/gateway/DNS, tested IP vs DNS.  
**Findings:** DNS resolution failing while IP connectivity OK.  
**Resolution:** Updated DNS settings and reconnected Wi-Fi.  
**Verification:** User can browse multiple sites and access email.
