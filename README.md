# Help Desk Runbooks (Tier 1)

A growing set of troubleshooting runbooks, decision trees, and ticket note templates built to practice real Tier 1 support workflows.

## Why this repo exists

- Practice structured troubleshooting instead of guesswork
- Build documentation habits used in real IT teams
- Prepare for Help Desk / IT Support interviews
- Keep portfolio work organized by skill area

## Main Focus

This repository is focused on Help Desk and IT Support documentation.

Core topics include:

- Windows troubleshooting
- Network connectivity issues
- Ticket documentation
- User support workflows
- Escalation notes
- Basic Active Directory support
- Office 365 support scenarios

## What's Inside

- **Runbooks:** step-by-step troubleshooting guides
- **Decision trees:** fast isolation of root cause
- **Ticket note templates:** clean documentation and escalation-ready notes
- **Supplemental skills:** related technical practice kept separate from core runbooks

## How to Use

1. Pick a runbook.
2. Follow checks in order.
3. Record findings like a ticket.
4. Verify the fix.
5. Document the result clearly.

## Portfolio Note

This repo is intentionally organized so Help Desk, cybersecurity, and data analysis work do not get mixed together without context. SQL practice is included only as a supplemental skill because it supports reporting, ticket analysis, asset audits, and future SOC/data analysis work.
